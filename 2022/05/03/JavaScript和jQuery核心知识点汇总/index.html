  • еҜ№иұЎзҡ„еҮ з§Қе®ҡд№үж јејҸпјҡ

  • еҜ№дәҺжҳҜиҮӘе·ұеҲӣе»әзҡ„пјҢж—ўеҗ«жңүж•°еӯ—и§’ж ҮпјҢеҸҲеҗ«жңүеӯ—з¬ҰдёІи§’ж Үзҡ„jsеҜ№иұЎиҝӣиЎҢйҒҚеҺҶж—¶дҪҝз”ЁеўһејәforеҫӘзҺҜиҝӣиЎҢйҒҚеҺҶгҖӮеўһејәforеҫӘзҺҜж—ўеҸҜд»ҘйҒҚеҺҶеҮәи§’ж Үдёәж•°еӯ—зҡ„еҖјд№ҹеҸҜд»ҘйҒҚеҺҶеҮәи§’ж Үдёәеӯ—з¬ҰдёІзҡ„еҖјгҖӮ

еүҚз«ҜйҒҮеҲ°BUGзҡ„еӨ„зҗҶжҖқи·Ҝвҳ…вҳ…вҳ…вҳ…вҳ…

еҰӮжһңдҪ еҲҡжҺҘи§ҰеүҚз«ҜзҹҘиҜҶпјҢд№ӢеүҚеҜ№е…¶дёҖж— жүҖзҹҘпјҢеңЁеүҚз«ҜйҒҮеҲ°й—®йўҳеҗҺжҖ»жҳҜжүҫеҫҲд№…иҝҳжүҫдёҚеҲ°й”ҷиҜҜеңЁе“ӘйҮҢпјҢйӮЈд№ҲпјҢејәзғҲе»әи®®дҪ жҢүз…§жҲ‘дёӢйқўи®Ізҡ„и§ЈеҶіBUGзҡ„жҖқи·ҜжқҘпјҢжӯҘйӘӨиҷҪз„¶е°‘гҖҒз®ҖеҚ•пјҢдҪҶжҳҜжңүз”ЁжІЎз”ЁпјҢдёҖиҜ•дҫҝзҹҘгҖӮ

  • ж ёеҝғжӯҘйӘӨжҖ»з»“пјҡдёүжӯҘиө°еҠ дёӨжҳҺзЎ®
  • еүҚз«Ҝй—®йўҳзҡ„дёҖиҲ¬дҪ“зҺ°пјҡйў„жңҹз»“жһңдёҺе®һйҷ…дёҚзӣёз¬Ұ
    • з¬¬дёҖжӯҘпјҡF12жү“ејҖи°ғиҜ•з•ҢйқўжҹҘзңӢжҳҜеҗҰжңүжҠҘй”ҷпјҢиӢҘжҠҘй”ҷзҠ¶жҖҒз Ғдёә404пјҢдё”жҠҘй”ҷдҝЎжҒҜдёӯеҢ…еҗ«пјҡhttp://127.0.0.1:8020/favicon.icoй“ҫжҺҘпјҢеҲҷеҝҪз•ҘиҜҘй”ҷиҜҜеҚіеҸҜгҖӮ
    • з¬¬дәҢжӯҘпјҡиӢҘи°ғиҜ•з•Ңйқўдёӯж— й”ҷиҜҜдҝЎжҒҜпјҢеҲҷи°ғиҜ•зӣёе…іеҮҪж•°пјҢжҹҘзңӢеҮҪж•°жҳҜеҗҰзңҹзҡ„иў«и°ғз”ЁдәҶпјҢеҰӮзӮ№еҮ»дәӢд»¶и°ғз”Ёзҡ„еҮҪж•°гҖӮеҰӮдҪ•жөӢиҜ•зӣёе…іеҮҪж•°пјҹеңЁеҮҪж•°зҡ„з¬¬дёҖиЎҢж·»еҠ дёҖдёӘalertпјҲпјүжҲ–иҖ…console.log()йғҪиЎҢгҖӮ
    • з¬¬дёүжӯҘпјҡиӢҘеҮҪж•°д№ҹзЎ®е®һиў«и°ғз”ЁдәҶпјҢдҪҶжҳҜйў„жңҹз»“жһңдёҺе®һйҷ…иҝҳжҳҜдёҚзӣёз¬ҰпјҢйӮЈд№ҲеҚҒжңүе…«д№қе°ұжҳҜд»Јз ҒйҖ»иҫ‘жңүй—®йўҳгҖӮ
    • дёҖжҳҺзЎ®пјҡжҳҺзЎ®иҮӘе·ұеҶҷзҡ„жҜҸдёҖжӯҘжҳҜд»Җд№Ҳж„ҸжҖқпјҢз„¶еҗҺеҶҚи·ҹзқҖд»Јз ҒдёҖжӯҘжӯҘзҡ„иҝӣиЎҢеҲҶжһҗпјҢиҝҷж ·еӨ§йғЁеҲҶзҡ„й”ҷиҜҜйғҪиғҪеӨҹиў«и§ЈеҶігҖӮ
    • дәҢжҳҺзЎ®пјҡжҳҺзЎ®зЁӢеәҸжөҒжҳҜжҖҺж ·зҡ„д»ҺдёҠеҫҖдёӢжү§иЎҢзҡ„гҖӮ

jQuery

еӯҰд№ jQueryиҝҷеқ—дёҖе®ҡиҰҒеј„жё…жҘҡjQueryеҜ№иұЎе’ҢJavaScriptеҺҹз”ҹеҜ№иұЎдёӨиҖ…еҲ°еә•еӯҳеңЁд»Җд№Ҳж ·зҡ„е…ізі»пјҢеҚғдёҮдёҚиҰҒе°ҶдёӨиҖ…ж··ж·ҶпјҢеӨҡеҶҷзӮ№д»Јз ҒпјҢзңӢдёӢдёӨиҖ…зҡ„еҢәеҲ«еҲ°еә•жҳҜд»Җд№ҲпјҢиӢҘжҳҜиҝҷйҮҢеј„дёҚжё…жҘҡпјҢйӮЈд№Ҳз”Ёиө·JavaScriptдёҺjQueryйҡҫе…Қдјҡй”ҷиҜҜзҷҫеҮәгҖӮ

JavaScriptеҺҹз”ҹеҜ№иұЎдёҺjQueryеҜ№иұЎзҡ„иҪ¬жҚўвҳ…вҳ…вҳ…вҳ…вҳ…

  • JavaScriptпјҡ
    • йҖҡиҝҮidиҺ·еҸ–дёҖдёӘеҜ№еә”зҡ„еҜ№иұЎ
      • var btn = document.getElementById("btn");
      • иҺ·еҸ–д№ӢеҗҺеҚідёәдёҖдёӘеҜ№еә”зҡ„е…ғзҙ еҜ№иұЎпјҢеҸҜд»ҘзӣҙжҺҘйҖҡиҝҮbtn.valueиҺ·еҸ–иҜҘеҜ№иұЎдёӯеҜ№еә”зҡ„valueеҖјгҖӮ
      • JavaScriptе…ғзҙ еҜ№иұЎиҪ¬jQueryеҜ№иұЎпјҡ$(btn)
      • з»‘е®ҡдәӢд»¶пјҡbtn.onclick = function(){}
    • йҖҡиҝҮclassиҺ·еҸ–еӨҡдёӘеҜ№еә”зҡ„еҜ№иұЎ
      • var tt = document.getElementsByClassName("tt");
      • иҺ·еҸ–д№ӢеҗҺдёәдёҖдёӘж•°з»„еҜ№иұЎпјҢйңҖиҰҒйҖҡиҝҮйҒҚеҺҶиҺ·еҸ–е…¶дёӯзҡ„жҜҸдёӘе…ғзҙ еҜ№иұЎпјҢд№ӢеҗҺжүҚиғҪиҺ·еҸ–иҜҘеҜ№иұЎдёӯеҜ№еә”зҡ„valueеҖјгҖӮ
      • JavaScriptе…ғзҙ еҜ№иұЎиҪ¬jQueryеҜ№иұЎпјҡ$(tt) гҖӮ
      • з»‘е®ҡдәӢд»¶пјҡеҝ…йЎ»е…ҲйҒҚеҺҶпјҢд№ӢеҗҺеҶҚз»‘е®ҡгҖӮ
  • jQueryпјҡ
    • йҖҡиҝҮidиҺ·еҸ–дёҖдёӘеҜ№еә”зҡ„jQueryеҜ№иұЎ
      • var btn = $("#btn");
      • иҺ·еҸ–д№ӢеҗҺдёәдёҖдёӘjQueryеҜ№иұЎпјҢиҰҒжғіиҺ·еҸ–иҜҘеҜ№иұЎдёӯзҡ„valueеҖјпјҢйӮЈд№Ҳе°ұеә”иҜҘз”ЁеҜ№еә”зҡ„jQueryзҡ„ж–№ејҸиҝӣиЎҢиҺ·еҸ–пјҢеҚіпјҡ$("#btn").val() гҖӮ
      • jQueryеҜ№иұЎиҪ¬JavaScriptе…ғзҙ еҜ№иұЎпјҡ$("#btn")[0]
      • з»‘е®ҡдәӢд»¶пјҡ$("#btn").click(function(){})
    • йҖҡиҝҮclassйҖүжӢ©еҷЁиҺ·еҸ–дёҖдёӘеҜ№еә”зҡ„jQueryеҜ№иұЎ
      • var tt = $(".tt");
      • иҺ·еҸ–д№ӢеҗҺдёәдёҖдёӘjQueryеҜ№иұЎпјҢиҖҢиҝҷдёӘjQueryеҜ№иұЎдёӯеҢ…еҗ«еӨҡдёӘеҺҹз”ҹJavaScriptе…ғзҙ еҜ№иұЎпјҢиӢҘжҳҜзӣҙжҺҘдҪҝз”Ёvar tt = $(".tt").val()пјҢйӮЈд№ҲеҸӘиғҪиҺ·еҸ–з¬¬дёҖдёӘе…ғзҙ еҜ№иұЎеҜ№еә”зҡ„valueеҖјпјҢиӢҘжҳҜиҰҒиҺ·еҸ–жүҖжңүзҡ„valueеҖјпјҢеҝ…йЎ»иҰҒеҜ№иҜҘjQueryеҜ№иұЎиҝӣиЎҢйҒҚеҺҶиҺ·еҸ–жҜҸдёӘе…ғзҙ еҜ№иұЎеҗҺеҶҚиҺ·еҸ–е…¶valueеҖјгҖӮдҪҶжҳҜпјҢиӢҘжҳҜеҜ№иҝҷдёҖз»„е…ғзҙ еҜ№иұЎиҝӣиЎҢдәӢд»¶зҡ„з»‘е®ҡж—¶пјҢзӣҙжҺҘйҮҮз”Ёtt.click(function(){})з»‘е®ҡеҚіеҸҜпјҢдёҚйңҖиҰҒйҒҚеҺҶиҺ·еҸ–жҜҸдёӘе…ғзҙ еҗҺеҶҚиҝӣиЎҢз»‘е®ҡгҖӮ
      • jQueryеҜ№иұЎиҪ¬JavaScriptе…ғзҙ еҜ№иұЎпјҡ$(".tt").each(function(){alert(this.value)})гҖӮ
      • з»‘е®ҡдәӢд»¶пјҡ$(".tt").click(function(){}) еҸҜиҝӣиЎҢдәӢд»¶зҡ„жү№йҮҸз»‘е®ҡгҖӮ

thisдёҺjQueryд№Ӣй—ҙзҡ„е…ізі»вҳ…вҳ…вҳ…вҳ…

еңЁдёҠйқўд»Јз Ғзҡ„дәӢд»¶з»‘е®ҡдёӯthisдёҺеҪ“еүҚjQueryеҜ№иұЎзҡ„е…ізі»пјҹ

  • йҖҡиҝҮдёҠиҝ°д»Јз ҒпјҢеҸҜд»Ҙе®һзҺ°еҜ№еҗ«жңүзұ»еҗҚдёәttзҡ„е…ғзҙ еҜ№иұЎиҝӣиЎҢзӮ№еҮ»дәӢд»¶зҡ„з»‘е®ҡпјҢдҪҶжҳҜз»‘е®ҡеҗҺеҮҪж•°зҡ„жү§иЎҢжҳҜеңЁзӮ№еҮ»дәӢд»¶еҸ‘з”ҹзҡ„жүҚжү§иЎҢзҡ„пјҢеҪ“зӮ№еҮ»ж—¶пјҢзӮ№еҮ»е“ӘдёӘе…ғзҙ иҠӮзӮ№йӮЈд№Ҳthisе°ұд»ЈиЎЁе“ӘдёӘе…ғзҙ еҜ№иұЎпјҢеӣ жӯӨпјҢиҜҫдёҠи®Ізҡ„и°Ғи°ғз”ЁеҮҪж•°пјҢйӮЈд№Ҳthisе°ұд»ЈиЎЁи°ҒеңЁжӯӨеӨ„д»Қз„¶йҖӮз”Ё

jQueryдёӯеҝ…дјҡAPIжҖ»з»“вҳ…вҳ…вҳ…вҳ…

з”ұдәҺAPIж–ҮжЎЈдёҠйқўи§ЈйҮҠзҡ„еҫҲиҜҰз»ҶпјҢиҖҢдё”жңүе…·дҪ“зҡ„е®һдҫӢеҸҜд»ҘжөӢиҜ•пјҢеӣ жӯӨеңЁиҝҷйҮҢжҲ‘е°ұдёҚеҶҚиөҳиҝ°пјҢеҸӘе°ҶйҮҚзӮ№APIеңҲеҮәпјҢе…·дҪ“и§ҒдёӢеӣҫгҖӮ

е…¶д»–жіЁж„ҸдәӢйЎ№вҳ…вҳ…вҳ…

  • JavaScriptдёӯиҝӣиЎҢеӯ—з¬ҰдёІеҶ…е®№жҜ”иҫғж—¶пјҢзӣҙжҺҘз”Ё==иҝӣиЎҢжҜ”иҫғеҚіеҸҜпјҢJavaScriptдёӯжІЎжңүequals()ж–№жі•пјҒпјҒ
  • JavaScriptдёӯеҸҜд»Ҙз”ЁLocation.href = вҖңURLзҪ‘еқҖвҖқ зҡ„ж–№ејҸиҝӣиЎҢи·іиҪ¬й“ҫжҺҘпјҢеҪ“зЁӢеәҸиө°еҲ°иҝҷйҮҢж—¶е°ұдјҡиҮӘеҠЁиҝӣиЎҢй“ҫжҺҘи·іиҪ¬гҖӮ
  • е…ідәҺеҜ№дёҖдёӘж Үзӯҫз»‘е®ҡеӨҡдёӘзӣёеҗҢдәӢд»¶зҡ„зҠ¶еҶөеҲҶжһҗ
    • еңЁж ҮзӯҫдёӯзӣҙжҺҘз»‘е®ҡдёӨдёӘзӣёеҗҢзҡ„дәӢд»¶пјҢйӮЈд№ҲжҜҸж¬Ўжү§иЎҢзҡ„ж—¶еҖҷеҸӘдјҡжү§иЎҢз¬¬дёҖдёӘз»‘е®ҡзҡ„дәӢд»¶
    • з”ЁеҺҹз”ҹjsзҡ„ж–№ејҸеҜ№дёҖдёӘж ҮзӯҫеҗҢж—¶з»‘е®ҡдёӨдёӘзӣёеҗҢзҡ„дәӢд»¶пјҢеҗҺз»‘е®ҡзҡ„дәӢд»¶е§Ӣз»ҲдјҡиҰҶзӣ–е…Ҳз»‘е®ҡзҡ„дәӢд»¶пјҲе…Ҳз»‘е®ҡзҡ„дәӢд»¶дёӯзҡ„ж–№жі•жҳҜдёҚдјҡжү§иЎҢзҡ„пјү
    • з”ЁjQueryзҡ„ж–№ејҸеҜ№дёҖдёӘж ҮзӯҫеҗҢж—¶з»‘е®ҡдёӨдёӘзӣёеҗҢзҡ„дәӢд»¶пјҢйӮЈд№ҲжҢүз…§з»‘е®ҡдәӢд»¶зҡ„е…ҲеҗҺйЎәеәҸеҺ»жү§иЎҢж–№жі•гҖӮ
